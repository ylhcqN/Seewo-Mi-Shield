# Seewo Mi Shield

Seewo Mi Shield 是一款基于 Mihomo （Clash Meta）内核开发的希沃（Seewo）集控系统监控屏蔽工具。通过精准域名分流与流量拦截，帮助用户有效阻断希沃校园设备运维管理系统对终端设备的远程监控、画面采集、行为管控等数据上报通道。

## 核心功能

| 功能模块 | 屏蔽内容 | 状态 |
|---------|---------|------|
| 远程巡课 | STUN 打洞服务、远程连接 IP | 可屏蔽 |
| 设备画面 | 屏幕画面上传、云存储同步 | 可屏蔽 |
| 设备管控 | IoT 设备管理、指令下发 | 可屏蔽 |
| 软件升级/文件传输 | 固件更新、文件分发 | 可屏蔽 |
| 软件安装 | 应用商店、第三方软件源 | 可屏蔽 |
| AC 部署 | 镜像仓库、容器注册中心 | 可屏蔽 |
| 希沃校园运维系统 | 设备状态上报、管理平台 | 可屏蔽 |
| 广告拦截 | 内置广告过滤规则集 | 可屏蔽 |

## 技术特性

- **fake-ip DNS 模式** - 使用 198.18.0.1/16 虚拟 IP 段，避免真实 DNS 泄露
- **域名嗅探 (Sniffer)** - 自动识别 TLS SNI / HTTP Host，精准匹配加密流量
- **TUN 透明代理** - 系统级流量接管，无需手动配置应用代理
- **分组灵活控制** - 每个功能模块独立分组，可选择 REJECT（屏蔽）或 DIRECT（放行）
- **规则自动更新** - 支持远程规则集定时同步

## 适用场景

- 学校机房、教室希沃白板/一体机的隐私保护
- 防止远程桌面、屏幕监控、摄像头调用
- 阻断静默软件安装与系统升级
- 防范设备位置、使用时长等元数据上报

## 快速使用

1. 安装支持 Mihomo 内核的客户端（如 Clash Verge Rev、FlClash、Surfboard，推荐 mihomo party 等）
2. 导入本配置文件
3. 在 proxy-groups 中将需要屏蔽的模块切换为 REJECT
4. 开启 TUN 模式（系统级拦截）或配置系统代理
5. **关闭 GUI 客户端自带的覆写或者改成配置文件一致！！！**
